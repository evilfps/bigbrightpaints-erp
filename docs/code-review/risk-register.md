# Central risk register

This register carries forward the findings from every completed area review plus the governance-review synthesis work. Each detailed entry keeps a consistent severity, category, affected-surface, finding, and evidence field so later remediation planning can trace the risk back to the originating review artifact.

Repeated themes (for example the inventory-accounting double-posting boundary) are intentionally preserved under each source review that found them. The goal of this register is traceability first, deduplication second.
