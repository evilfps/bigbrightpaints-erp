# AGENTMAP.md

Primary agent instruction map for humans.

Canonical runtime file for Codex remains `AGENTS.md`.
This file exists as an explicit alias so it is easy to identify in large repos.

## Use
- Human readers: start here, then open `AGENTS.md`.
- Codex/automation: always uses `AGENTS.md`.

## Canonical Files
- `AGENTS.md` (runtime instruction map)
- `docs/INDEX.md`
- `docs/agents/CATALOG.md`
- `docs/agents/PERMISSIONS.md`
- `docs/agents/WORKFLOW.md`

## Why both files exist
- `AGENTS.md` is required for instruction discovery.
- `AGENTMAP.md` is a stable, human-visible naming convention for this repo.
